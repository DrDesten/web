<div id="main-content"></div> <!-- Creating Div as Section marker --> <Paragraph title="Stracciatella Shaders" id="stracc-shader"> <p class="paragraph"> Stracciatella is a shaderpack which aims to enhance the vanilla experience without sacrificing performance or changing the vanilla style. It allows for extensive customization in almost all aspects of the game, like lightmap or sky colors. It also fixes vanillas broken fog and adds some new features which go nicely with the vanilla style. This would for example be rain refraction, which distorts the image behind raindrops as well as rain puddles which show the vanilla water texture instead of realistic reflections.<br> All features of this shader are completely optional, so you can pick and choose how <b>you</b> want your vanilla experience. You can download Stracciatella on <a href="http://www.curseforge.com/minecraft/customization/stracciatella-shaders" target="_blank" rel="noopener noreferrer">Curseforge</a>. </p> </Paragraph> <Background /> <script src="style.js"></script> </body> </html>
