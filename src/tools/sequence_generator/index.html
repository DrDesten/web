Sequence Generator

This Tool allows you to generate nonlinear (and linear) number sequences.

Options
Sequence Length: Amount of numbers in a sequence
Start / End: Sequence Start and End (inclusive)
Linearity: Skews the numbers no either emphasize lower or higher numbers. Positive values emphasize lower numbers, negative values emphasize higher numbers. 0 is linear.
Human Rounding: Adds a digit and rounds so that it's either 5 or 0, looks nicer.
