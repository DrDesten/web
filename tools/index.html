Tools

Tools that come in handy

In the course of shader developement I needed some external tools to get things done. Following you can find most of them, so you don't have to go and code them for yourself.

Gaussian Kernel Calculator

Allows you to calculate sample weights for a binomial distribution. Useful for gaussian blurs. Take me there!

Vogel Disk Sample Calculator

Calculates samples on a disk using the vogel method. Useful for creating realistic circlular Depth-of-Field blurs. Take me there!
